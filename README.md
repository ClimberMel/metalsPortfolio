# metalsPortfolioGUI

An app to track metals purchases and the portfolio balances.

## Prerequisites

* Python 3
* metalsPortfolioGUI.py
* file.py
* prices.json
* portfolio.json

## Built With

* [VisualStudioCode](https://code.visualstudio.com/)
